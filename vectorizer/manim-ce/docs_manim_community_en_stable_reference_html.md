# Reference Manual [¶](https://docs.manim.community/en/stable/reference.html\#reference-manual "Link to this heading")

This reference manual details modules, functions, and variables included in
Manim, describing what they are and what they do. For learning how to use
Manim, see [Tutorials](https://docs.manim.community/en/stable/tutorials/index.html). For a list of changes since the last release, see
the [Changelog](https://docs.manim.community/en/stable/changelog.html).

The pages linked to here are currently a work in progress.
